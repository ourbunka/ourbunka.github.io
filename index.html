PRIVACY POLICY FOR OURBUNKA APP

This privacy policy governs your use of the software application ourbunka (“Application”) by OURBUNKA MOTION. The Application allow user to

⚪ to stay up to date with latest product and services offered by OURBUNKA MOTION.

⚪ manage key and key signing with In App Chimera Auth Key Manager.


What information does the Application collect/obtain and how is it used?

For User without an ourbunka account, this Application does not collect or obtain user information.

User are not required to register account to use this app.

For User with an ourbunka account, this Application does collect

⚪ pseudonymous In App Chimera Auth public key for account creation, authentication and verification purposes.

⚪ signatures of signed messages from OURBUNKA MOTION digital products via in app Chimera Auth Signing/Scan QR features, for account authentication purposes.

ourbunka server automatically create a new ourbunka account for user and collect data as listed above when user "CONNECT", i.e login, for the first time.

Collected Data are securely transfer and store in ourbunka server for account creation and verification purposes and can be deleted when user Request to "Delete Account"(coming soon) in app.


User Provided information

This Application does not collect nor required user with or without ourbunka account to provide any form of personal infomation. This App do not ask or required user to provide personal information such as email, phone number, username or any informations that can tied to your real identity when creating an ourbunka app account, as this app only need a unique pseudonymous public key generated via in app Chimera Auth to create an ourbunka app account. ourbunka app are designed to be private by default.


In App Chimera Auth

In App Chimera Auth Key Manager allow user to

⚪ managing chimera auth private key,

⚪ generate chimera auth private key on device LOCALLY and ONLY store on user device locally,

Generated private key are not and should not shared with anyone. You are required to keep your generated chimera auth private key safe and secured, or you will lose access to all your OURBUNKA MOTION digital product accounts once you lost your chimera auth private key. OURBUNKA MOTION digital products account are designed to be secured by public key cryptography, each account is only be accessible by one unique private key, and can not be changed. OURBUNKA MOTION do not have abilities to help you recover your private key or accounts.

⚪ "Scan QR" via android google play services barcode api

⚪ signing message from OURBUNKA MOTION digital products using public key cryptography for authentication purposes.

Only signing signature are collected by OURBUNKA when using this features, for authentication purposes.

Signature are generated using your chimera auth private key locally on devices.

⚪ recover key with existing chimera auth private key.

Chimera Auth private key provided by user/you during recovering key are not shared with anyone, including OURBUNKA MOTION. Provided Key are processed locally and stored locally on your devices.

NOTE: YOU SHOULD ONLY USE PRIVATE KEY GENERATED FROM OURBUNKA IN APP CHIMERA AUTH FEATURE WHEN RECOVERING PRIVATE KEY IN CHIMERA AUTH "RECOVERING KEY". DO NOT USE PRIVATE KEY GENERATED FROM OTHER APPS/WEBSITES.

Chimera Auth are designed only to be used with ourbunka motion digital products, do not use it with other websites or apps.

Do Not use private key generated from ourbunka Chimera Auth in other apps/websites, your chimera auth private key could be stolen by other malicious app/websites.

Any Loses occured when using ourbunka Chimera Auth outside of designed/intended purposes are on you.


Does the Application collect precise real time location information of the device?

This Application does not collect precise information about the location of your mobile device.
